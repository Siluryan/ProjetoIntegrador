**Dia 1**

Instalação:
- Python
- Pip
- Django

Comandos:
```bash
- python --version ou python -V
- pip --version
- python -m pip install --upgrade pip
- pip install django
- django-admin startproject nome_do_projeto
- python3 manage.py startapp nome_da_app
```
